# 💫 About Me:
Hi! I'm a passionate Fullstack Developer who enjoys building modern web applications end to end, from clean and intuitive user interfaces to scalable backend systems. I have experience working with technologies like React, Next.js, Tailwind CSS, Node.js, Express, PostgreSQL, and MongoDB, and I love turning complex data into simple, interactive, and meaningful user experiences. Recently, I built a data analytics platform that integrates external APIs and presents insights through a responsive dashboard, sharpening my skills in both frontend and backend development. I'm currently focused on improving my knowledge in system design, scalability, and DevOps, with the goal of becoming a Software Engineer who builds impactful and reliable products.<br>
